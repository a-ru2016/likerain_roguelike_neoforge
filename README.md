"# Likerain Roguelike (NeoForge)

Minecraft 1.21.1 (NeoForge) 環境において、本格的なローグライク（不思議のダンジョン風）のゲーム進行システムを導入する Mod です。
マルチプレイに完全対応しており、プレイヤーの獲得アイテムや経験値量に応じた難易度の自動上昇、転生システム、夜間死亡制限、白夜とボス、全滅時のリスタートペナルティなどを特徴としています。

> [!NOTE]
> 本 Mod の内部 Mod ID は `timeattackroguelike` です。

---

## 🎮 主要なゲームシステムとフロー

### 1. ゲームの開始と難易度設定
- **ゲーム開始**: ログイン時に全プレイヤーに配布される**「開始の本」**を使用することでゲームを開始します。この本は使用すると消滅します。
- **設定GUI**: 本を使用すると設定画面が開き、以下を設定できます。
  - **スターターキット**: 初期装備やアイテムを選択できます。
  - **難易度（熱度）調整**: スマブラのソロモードや3DSパルテナの鏡のようなメーター（スライダー）で調整します。
- **難易度（熱度）の影響**:
  - 難易度を上げるほど、昼の時間が短くなり、夜の時間が長くなります。
  - 難易度が高いほど、転生時に獲得できるポイントの倍率がアップします。

### 2. 夜間の脅威と朝のデバフ
- **夜間の敵増加**: 夜間は時間の経過とともに、出現するモンスターの数が徐々に増加します。朝を迎えることでこの増加分はリセットされます。
- **朝のデバフ付与**: 
  朝（日の出のタイミング）が来るたびに、プレイヤーのそれまでの「累計アイテム獲得数」および「獲得経験値量」に応じて、デバフが付与され、モンスターが強化されます。
  - **アイテム獲得過多**: ポーション効果（鈍化等）
